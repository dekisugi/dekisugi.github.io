# シンタックスハイライトについて

当サイトでは、シンタックスハイライトにRougeを使用しています。


## rouge ディレクトリ内のCSSファイルについて

下記コマンドで吐き出したファイルです。

    rougify style github > rouge.css

Rougeのライセンスは「[LICENSE](./LICENSE)」ファイルをご覧下さい。
